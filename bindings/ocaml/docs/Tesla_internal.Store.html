Module Tesla_internal.Store

module Store: sig .. end
A storage container for one or more Tesla_internal.Class.t values. There may be one tesla_store for each thread (for storing thread-local automata) plus a single global tesla_store.

Retrieve the tesla_store for a context (e.g., a thread). If the tesla_store does not exist yet, it will be created.

Allocate a null pointer to a store, suitable for passing into a function that mutates the pointer to point to a store.
