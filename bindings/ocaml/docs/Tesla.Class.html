Module Tesla.Class

module Class: sig .. end
A description of a TESLA automaton, which may be instantiated a number of times with different names and current states.
type t 
type id = int
A client-generated handle (a small integer, used as an index into an array internal to libtesla.
val get : store:Tesla.Store.t ->
       id:id -> name:string -> descr:string -> t
Retrieve (or create) a Tesla.Class.t from a Tesla.Store.t. Once the caller is done with the Tesla.Class.t, Tesla.Class.put must be called to release the lock. See Tesla.with_class for a wrapper that enforces this protocol.
val put : t -> unit
Release the resources that were associated with the Tesla.Class.t that was retrieved via Tesla.Class.get.
