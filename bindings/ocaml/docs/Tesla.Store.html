Module Tesla.Store

module Store: sig .. end
A storage container for one or more Tesla.Class.t values. There may be one Tesla.Store.t for each thread (for storing thread-local automata) plus a single global Tesla.Store.t that is retrieved using Global.
type t 
val get : ctx:Tesla.ctx -> size:int -> instances:int -> t
Retrieve the Tesla.Store.t for a context (e.g., a thread). If the Tesla.Store.t does not exist yet, it will be created.
val size : t -> int
Retrieve the maximum size that this Tesla.Store.t was created with.
val instances : t -> int
Retrieve the maximum number of instances that this Tesla.Store.t was initialized to support.
