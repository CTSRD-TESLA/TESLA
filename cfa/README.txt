cfa
===

Translates automata definition (.spl) files into tesla, ocaml, dot,
promela files.

Contents
--------
*.ml, *.mli, *.mly, *.mll: source files
Makefile, OCamlMakefile: build files
*.spl: example automata definitions

Requirements
------------
make, ocaml (ocamldep.opt, ocamlyacc, ocamllex, ocamlc.opt)

Building
--------
make

Results
-------
splc binary
